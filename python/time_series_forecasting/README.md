# Time Series Analysis — Avocado Price Trends

## Overview

Проект посвящён анализу временных рядов на примере данных о средних ценах на авокадо от Hass Avocado Board. Основная задача — выявление трендов и сглаживание колебаний с помощью оконных функций.

**Цель проекта** — продемонстрировать навыки работы с временными рядами, скользящими средними, экспоненциальным сглаживанием и интерактивной визуализацией.

## Business Tasks Solved

### 1. Data Aggregation by Date
- Группировка данных по датам
- Расчёт средневзвешенной цены авокадо для каждой даты
- Результат: временной ряд средних цен

### 2. Rolling Window Analysis (Скользящее среднее)
- Расчёт скользящего среднего с окном 3
- Определение максимального значения (округлено до 2 знаков)
- Сравнение окон 2, 4, 10, 50 — визуализация влияния размера окна на сглаживание

### 3. Exponential Weighted Moving Average (EWM)
- Расчёт экспоненциального скользящего среднего с параметром span=2
- Более чувствительный к последним изменениям метод сглаживания

### 4. Interactive Visualization with Plotly
- Фильтрация данных: только органические авокадо в Чикаго
- Расчёт скользящего среднего с окном 4
- Построение **интерактивного линейного графика** (plotly.express)
- Возможность наведения на точки для точных значений в конкретные даты

### 5. Price Segmentation with pd.cut()
- Разбиение цен на 2 группы: выше среднего и ≤ среднего
- Создание категориальной колонки `AveragePrice_label`
- Подсчёт количества записей в каждой группе

## Key Insights (Бизнес-выводы)

| Вывод | Бизнес-интерпретация |
|-------|----------------------|
| Скользящее среднее с окном 3 сглаживает недельные колебания | Позволяет увидеть общий тренд без шума |
| При увеличении окна (50) график становится почти линейным | Для долгосрочного прогнозирования можно использовать большое окно |
| Экспоненциальное сглаживание (EWM) быстрее реагирует на изменения | Подходит для отслеживания резких скачков цен |
| Цены на органические авокадо в Чикаго распределены неравномерно | Большая часть значений ≤ среднего — значит, есть небольшое количество аномально высоких цен |
| Интерактивный график позволяет точно определить цену в любой дату | Удобно для ad-hoc запросов от бизнеса |

## Key Pandas & Visualization Concepts

| Метод / Приём | Где использован |
|---------------|----------------|
| `pd.read_csv(parse_dates=['Date'])` | Загрузка с преобразованием дат |
| `groupby().mean()` | Агрегация по датам |
| `rolling(window).mean()` | Скользящее среднее |
| `ewm(span).mean()` | Экспоненциальное сглаживание |
| `query()` | Фильтрация по типу и региону |
| `set_index()` | Дата в индекс для rolling |
| `round()` | Округление результатов |
| `pd.cut(bins, labels)` | Бинаризация числовых значений |
| `value_counts()` | Подсчёт по категориям |
| `sns.lineplot()` | Визуализация сравнения окон |
| `plt.subplots()` | Множественные графики |
| `plotly.express.line()` | Интерактивный график |
