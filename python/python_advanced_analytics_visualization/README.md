# 📊 Advanced Calculations & Visualization (Python, Pandas)

## Overview
В данном проекте решены аналитические задачи с использованием библиотеки pandas, направленные на исследование пользовательского поведения, расчёт продуктовых метрик и визуализацию данных.

Основной фокус — применение группировок, агрегаций, фильтрации и построение графиков для получения бизнес-инсайтов.
Задания взяты с курса KarpovCourses.
---

## Цели проекта
В рамках проекта были выполнены следующие задачи:

- анализ пользовательской активности  
- расчёт ключевых продуктовых метрик  
- сегментация пользователей  
- выявление закономерностей в данных  
- визуализация результатов анализа  

---

## Ключевые аналитические задачи

### 1. Анализ распределения данных
- исследование структуры датасета  
- анализ типов данных и пропусков  
- первичный EDA  

### 2. Группировка и агрегация
- расчёт метрик по группам (например, пользователи / продукты)  
- использование `.groupby()` и агрегирующих функций:  
  - mean  
  - sum  
  - count  

### 3. Фильтрация данных
- отбор данных по условиям с помощью `.query()`  
- анализ только релевантных наблюдений  

### 4. Расчёт метрик
- средние значения  
- суммарные показатели  
- распределения  
- частотный анализ (`value_counts`)  

### 5. Сегментация пользователей
- разделение пользователей по поведению  
- выявление активных и менее активных групп  

### 6. Визуализация данных
- построение графиков для анализа:  
  - распределения  
  - сравнений между группами  
- использование визуализации для интерпретации результатов
