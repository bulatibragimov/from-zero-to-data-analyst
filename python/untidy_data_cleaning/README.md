## Part 2. Обработка данных телемаркетинга

### Задача
Проверить корректность подключений продуктов через телемаркетинг, объединив:
- Файлы продаж (`tm_sales_1/2/3.csv`)
- Логи подключений (`prod_activations_logs.csv`)

### Особенности данных (которые были обработаны)
| Проблема | Решение |
|----------|---------|
| Пропущенные `SUBS_ID` | `dropna(subset=['SUBS_ID'])` |
| Отсутствует префикс `id` в начале | `apply(lambda x: 'id' + x if not x.startswith('id') else x)` |
| Разный порядок колонок в файлах | `pd.concat()` справляется автоматически |
| Даты в формате `день-месяц-год` | `pd.to_datetime(dayfirst=True)` |
| Отключение < 5 минут после подключения | Фильтрация по `END_DTTM - START_DTTM > 5 минут` |

### Бизнес-логика
Продажа **НЕ засчитывается**, если отключение произошло **меньше чем через 5 минут** после подключения.

### Результат
Сохранён CSV-файл `df_final.csv` (разделитель `;`) с корректными подключениями.

**SUBS_ID в порядке возрастания:** `id14646, id18768, id19735, id19903, ...` (подставьте из вашего кода)
