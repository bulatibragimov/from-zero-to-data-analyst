# Final Project — Wholesale Audio Equipment Analytics

## Overview

Финальный проект модуля Python. Комплексный анализ данных оптовых продаж аудиотехники компании «Карпов Саунд».

**Задача:** собрать данные из разрозненных источников (файловая система), провести полный анализ и подготовить выводы для руководства о ключевых метриках, аномалиях, эффективности брендов и менеджеров.

**Результат:** дашборд ключевых метрик, выявление причин аномального роста заказов, рекомендации по оптимизации ассортимента и оценка эффективности сотрудников.

## Business Tasks Solved

### 1. Data Collection & Preparation
- Обход файловой системы через `os.walk()`
- Сбор данных из 4603+ строк заказов, 346 статусов, 1677 товаров
- Добавление колонок: `date` (из имени папки), `manager` (из имени папки), `category` (из папки продукта)
- Сохранение чистых данных в CSV

### 2. Time Series Analysis (Заказы по дням)
- Определён день с максимальным числом заказов: **2024-03-14**
- Выявлен аномальный день: **2024-03-13** — нулевое количество подтверждённых заказов
- Построены графики динамики заказов по дням с разбивкой по статусам

### 3. Anomaly Investigation (Почему произошёл скачок?)
**Гипотеза:** сбой в CRM → все заказы 13 марта отменились → клиенты переоформили их 14 марта.

**Проверка:**
- Заказы 13 марта (canceled) сопоставлены с заказами 14 марта (confirmed)
- Критерии совпадения: клиент, менеджер, количество уникальных товаров, общее количество товаров
- **Результат:** 67% заказов 14 марта повторяют отменённые заказы 13 марта → гипотеза подтверждена

## Key Pandas & Visualization Concepts

| Метод / Приём | Где использован |
|---------------|----------------|
| `os.walk()` | Сбор данных из вложенных папок |
