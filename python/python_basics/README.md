# Python Basics

## Overview

В этом разделе рассматриваются базовые конструкции Python, необходимые для начала работы с данными.

Основное внимание уделено переменным, базовым типам данных и работе со списками.

---

# Topics Covered

## Variables

Переменные используются для хранения данных и управления логикой программы.

Хорошо подобранные имена переменных делают код понятным и удобным для чтения.

---

## Data Types

Были рассмотрены основные типы данных:

* int (целые числа)
* float (дробные числа)
* str (строки)
* bool (логические значения)

---

## Lists

Списки позволяют хранить несколько значений в одной переменной.

Мы научились:

* создавать списки
* получать элементы по индексу
* использовать срезы
* применять функции:

  * len()
  * sum()
  * min()
  * max()
  * round()

---

## List Methods

Основные методы списков:

* append(), insert(), extend() — добавление элементов
* remove(), pop() — удаление
* index(), count() — поиск
* sort(), reverse() — изменение порядка
* copy() — копирование
* clear() — очистка списка

---

## Mutability

Списки являются изменяемыми объектами.

Важно понимать разницу между:

* копией списка
* ссылкой на тот же объект

Использование `.copy()` помогает избежать ошибок при работе с данными.
