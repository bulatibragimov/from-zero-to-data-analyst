# Control Flow and String Processing

## Overview

В этом разделе рассматриваются базовые конструкции управления логикой в Python: условия, циклы и работа со строками.

Эти инструменты позволяют анализировать данные, фильтровать значения и строить бизнес-логику обработки информации.

---

# Topics Covered

## Conditional Statements

Условия (`if`, `elif`, `else`) используются для принятия решений в коде на основе заданных правил.

---

## Loops

Циклы позволяют выполнять повторяющиеся операции:

* `for` — перебор элементов
* `while` — выполнение до выполнения условия
* `break` и `continue` — управление выполнением цикла

---

## String Processing

Строки активно используются в аналитике для обработки текстовых данных.

Основные методы:

* `find()`
* `strip()`
* `replace()`
* `startswith()`
* `split()`
* `join()`

---

## String Formatting

Форматирование строк позволяет создавать отчёты:

* f-строки
* `.format()`

---

## Iteration Over Collections

Работа с:

* списками
* словарями
* множествами

с помощью циклов и генераторов.
