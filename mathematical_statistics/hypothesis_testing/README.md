# Проверка статистических гипотез

## Теория

### Нормальное распределение

Самое важное распределение в статистике. Многие природные и социальные явления распределены нормально.

**Z-стандартизация** — приведение к стандартному нормальному распределению со средним 0 и стандартным отклонением 1:

$$ z = \frac{x - \mu}{\sigma} $$

### Центральная предельная теорема (ЦПТ)

**Ключевой вывод:** при увеличении размера выборки распределение выборочных средних **стремится к нормальному**, независимо от исходного распределения.

- Среднее этого распределения совпадает со средним генеральной совокупности
- Стандартная ошибка (SE) = $\frac{\sigma}{\sqrt{n}}$ (отражает точность оценки)

### Статистический вывод

| Понятие | Описание |
|---------|----------|
| **Нулевая гипотеза (H₀)** | То, что мы проверяем (обычно "нет эффекта", "разницы нет") |
| **Альтернативная гипотеза (H₁)** | То, с чем сравниваем ("есть эффект", "есть разница") |
| **p-value** | Вероятность получить наблюдаемые данные (или более экстремальные), если H₀ верна |
| **Статистическая значимость** | p-value < α (обычно α = 0.05) — отвергаем H₀ |

### Важное замечание о p-value

p-value **НЕ** показывает:
- Вероятность того, что H₀ верна
- Размер эффекта
- Практическую значимость

**Что показывает:** насколько данные противоречат H₀.

## Практический анализ рекламных кампаний

### Данные

Рекламные кампании в Facebook с метриками:
- `impressions` — показы
- `clicks` — клики
- `spent` — расходы
- `approved_conversion` — покупки

### Метрики эффективности

#### 1. CTR (Click-Through Rate) — кликабельность

$$ CTR = \frac{clicks}{impressions} $$

**Что показывает:** насколько реклама привлекает внимание.

#### 2. CPC (Cost-Per-Click) — стоимость клика

$$ CPC = \frac{spent}{clicks} $$

**Что показывает:** экономическая эффективность (чем ниже, тем лучше).

#### 3. Conversion Rate (CR) — конверсия из клика в покупку

$$ CR = \frac{approved\_conversion}{clicks} $$

**Что показывает:** качество трафика и продукта.

## Связь с проверкой гипотез

На практике эти метрики используются для проверки гипотез:

| Гипотеза | Проверка |
|----------|----------|
| H₀: CTR кампании A = CTR кампании B | t-test или z-test для пропорций |
| H₀: CPC для мужчин = CPC для женщин | двухвыборочный t-test |
| H₀: конверсия не изменилась после изменений | тест на равенство пропорций |
