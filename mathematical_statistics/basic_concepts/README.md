# Основы статистики — описательная статистика

## Что я изучил

### 1. Генеральная совокупность и выборка

| Термин | Определение |
|--------|-------------|
| **Генеральная совокупность (ГС)** | Все возможные объекты, представляющие интерес (например, все клиенты бизнеса) |
| **Выборка** | Часть генеральной совокупности, которую анализируем на практике |
| **Репрезентативность** | Выборка должна быть максимально похожа на ГС по своим характеристикам |

### 2. Типы переменных

| Тип | Что это | Пример |
|-----|---------|--------|
| **Количественные** | "Сколько?" | Возраст, цена, количество |
| └ Непрерывные | Любое значение в интервале | Рост, время |
| └ Дискретные | Целые числа | Количество детей, число заказов |
| **Качественные** | "Что?" | Цвет, бренд, жанр |
| └ Категориальные | Без порядка | Страна, тип авокадо |
| └ Порядковые | С порядком | Оценки (отлично, хорошо, плохо) |

### 3. Распределения

| Тип | Описание |
|-----|----------|
| **Эмпирическое** | Реальное распределение, которое мы видим в данных |
| **Теоретическое** | Идеальная модель (нормальное, биномиальное и т.д.) |

### 4. Меры центральной тенденции

| Мера | Что показывает | Когда использовать |
|------|----------------|---------------------|
| **Среднее (mean)** | Сумма / количество | Когда нет сильных выбросов |
| **Медиана (median)** | Серединное значение | При наличии выбросов |
| **Усечённое среднее** | Среднее без крайних значений | Компромисс между средним и медианой |
| **Мода (mode)** | Самое частое значение | Для категориальных данных |

### 5. Квантили

Делят распределение на равные части:
- **Медиана** — 50% квантиль
- **Квартили** — 25%, 50%, 75%
- **Перцентили** — 1%, 2%, ..., 99%

### 6. Меры разброса

| Мера | Формула | Что показывает |
|------|---------|----------------|
| **Размах (range)** | max - min | Полный интервал данных |
| **Межквартильный размах (IQR)** | Q3 - Q1 | Разброс "средней части" данных |
| **Дисперсия (variance)** | среднее квадратов отклонений | Разброс в квадратных единицах |
| **Стандартное отклонение (std)** | √дисперсии | Разброс в исходных единицах |
| **MAD (медианное абсолютное отклонение)** | медиана \|xᵢ - медиана\| | Устойчиво к выбросам |

### 7. Визуализация распределений

| Тип графика | Что показывает | Когда использовать |
|-------------|----------------|---------------------|
| **Столбиковая диаграмма (bar plot)** | Частоты категорий | Категориальные данные |
| **Гистограмма (histogram)** | Распределение непрерывных данных | Количественные данные |
| **График плотности (KDE)** | Сглаженная гистограмма | Сравнение распределений |
| **Боксплот (box plot)** | Медиана, квартили, выбросы | Сравнение групп |
| **Скрипичный график (violin plot)** | Боксплот + плотность | Более детальное сравнение групп |
| **Пчелиный рой (swarm plot)** | Каждая точка отдельно | Маленькие выборки |

Файл `descriptive_statistics.ipynb` содержит анализ видеоигр:
