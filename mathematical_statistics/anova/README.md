# Сравнение нескольких средних — ANOVA (A/B/C-тест)

## Теория

### Дисперсионный анализ (ANOVA)

ANOVA — это обобщение t-теста для сравнения средних **трёх и более выборок**.

```python
import scipy.stats as st

# Однофакторный ANOVA
st.f_oneway(group_A, group_B, group_C)
```

### Недостаток ANOVA

ANOVA может обнаружить, что **различия есть**, но **не скажет**, какие именно группы отличаются друг от друга.

**Решение:** попарные сравнения (t-тест + поправки).

### Проблема множественных сравнений

Чем больше попарных сравнений, тем выше вероятность ошибки I рода (ложноположительный результат).

$$ \text{Вероятность хотя бы одной ошибки} = 1 - (1 - \alpha)^m $$

где m — количество сравнений.

### Поправки для множественных сравнений

| Метод | Подход | Когда использовать |
|-------|--------|---------------------|
| **Бонферрони** | α / m | Самый консервативный |
| **Холма** | Пошаговая корректировка | Менее консервативный |
| **Тьюки (HSD)** | Специально для всех попарных сравнений | После ANOVA |
| **Геймса-Хоувелла** | Для неравных дисперсий | Альтернатива Тьюки |

### Многофакторный ANOVA

Проверяет влияние **нескольких факторов** (группа + сегмент) и их **взаимодействие**.

```python
import pingouin as pg

pg.anova(data=df, dv="events", between=["group", "segment"])
```

## Практика: Тест 1 — изображения блюд

### Дизайн эксперимента

| Группа | Описание |
|--------|----------|
| **A** | прямоугольные изображения 16:9 |
| **B** | квадратные изображения |
| **C** | прямоугольные изображения 12:4 |

### Метрика

`events` — количество заказанных блюд за период.

---

### Результаты

#### 1. Проверка гомогенности дисперсий (тест Левена)

$$ p\text{-value} = 0.13 > 0.05 $$

Дисперсии **гомогенны** (равны) — ANOVA подходит.

#### 2. Проверка нормальности

| Группа | p-value | Распределение |
|--------|---------|---------------|
| A | <0.05 | Отличается от нормального |
| B | <0.05 | Отличается от нормального |
| C | <0.05 | Отличается от нормального |

*QQ-графики показывают умеренные отклонения. При больших выборках это не критично.*

#### 3. Однофакторный ANOVA

$$ F\text{-статистика} = 3.79, \quad p\text{-value} = 0.023 $$

**p-value < 0.05** → между группами есть статистически значимые различия.

#### 4. Попарные сравнения (критерий Тьюки)

| Сравнение | p-value | Различия |
|-----------|---------|----------|
| A vs B | >0.05 | Нет |
| A vs C | <0.05 | **Есть** |
| B vs C | >0.05 | Нет |

---

### Вывод по Тесту 1

Статистически значимые различия только между **группой A (16:9) и группой C (12:4)**.

**Рекомендация:** выбирать между A и C, ориентируясь на бизнес-метрики (не статистические).

## Практика: Тест 2 — обновление кнопки заказа

### Дизайн эксперимента

| Фактор | Уровни |
|--------|--------|
| **group** | control (старая кнопка), test (новая) |
| **segment** | high (активные), low (неактивные) |

### Метрика

`events` — количество заказанных блюд.

---

### Результаты

#### 1. Визуализация

Распределение `events` показывает различия между группами и сегментами.

#### 2. Описательные статистики

| group | segment | mean | median | std |
|-------|---------|------|--------|-----|
| control | high | ... | ... | ... |
| control | low | ... | ... | ... |
| test | high | ... | ... | ... |
| test | low | ... | ... | ... |

#### 3. Многофакторный ANOVA (pingouin)

| Источник | p-value | Значимость |
|----------|---------|------------|
| group | <0.05 | ✅ есть |
| segment | <0.05 | ✅ есть |
| group × segment (взаимодействие) | >0.05 | ❌ нет |

#### 4. Визуализация взаимодействия (pointplot)

График показывает, что эффект новой кнопки одинаков для обоих сегментов (линии параллельны).

---

### Вывод по Тесту 2

- Новая кнопка **статистически значимо** увеличивает число заказов
- Активные пользователи (high) заказывают больше, чем неактивные (low)
- Взаимодействия между группой и сегментом **нет** — эффект одинаков

**Рекомендация:** внедрить новую кнопку для всех пользователей ✅

## Итоговая таблица методов

| Задача | Метод | Функция |
|--------|-------|---------|
| Сравнить 2 группы | t-тест | `sts.ttest_ind()` |
| Сравнить 3+ группы | ANOVA | `sts.f_oneway()` |
| Проверить равенство дисперсий | Тест Левена | `sts.levene()` |
| Проверить нормальность | Тест Д'Агостино-Пирсона | `sts.normaltest()` |
| Попарные сравнения (равные дисперсии) | Тьюки HSD | `pg.pairwise_tukey()` |
| Попарные сравнения (неравные дисперсии) | Геймс-Хоувелл | `pg.pairwise_gameshowell()` |
| Многофакторный ANOVA | Двухфакторный ANOVA | `pg.anova()` |
| Визуализация нормальности | QQ-график | `pg.qqplot()` |
