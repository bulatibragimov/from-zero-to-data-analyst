# Сравнение средних значений — t-тест (A/B-тест)

## Теория

### t-тест (критерий Стьюдента)

Используется для сравнения средних двух выборок и оценки **статистической значимости** различий.

### Основа t-теста

- Центральная предельная теорема (ЦПТ)
- Поправка на неточность измерения стандартного отклонения

### Допущения t-теста

| Допущение | Важность | Комментарий |
|-----------|----------|-------------|
| **Независимость наблюдений** | Строгое | Обязательно |
| **Отсутствие выраженных выбросов** | Строгое | Влияют на среднее |
| Нормальность распределения | Не критично | При больших выборках |
| Равенство дисперсий | Не критично | Есть t-тест Уэлча |

### Формулы в Python

```python
import scipy.stats as sts

# t-тест
sts.ttest_ind(test, control)

# t-тест Уэлча (если дисперсии неравны)
sts.ttest_ind(test, control, equal_var=False)
```

### Вспомогательные тесты

| Тест | Назначение |
|------|------------|
| Тест Левена / Брауна-Форсайта | Равенство дисперсий |
| Тест Шапиро-Уилка / Д'Агостино-Пирсона | Нормальность |
| QQ-график | Визуальная проверка нормальности |

## Практический A/B-тест: алгоритм доставки пиццы

### Задача бизнеса

К вечеру скорость доставки падает (курьеры уходят домой, заказы растут). DS-команда разработала новый алгоритм: курьеры доставляют последние заказы **«по пути домой»**.

### Дизайн эксперимента

| Группа | Что получает |
|--------|--------------|
| **Контроль (control)** | Старый алгоритм |
| **Тест (test)** | Новый алгоритм "по пути домой" |

### Метрика

`delivery_time` — время доставки заказа в минутах.

---

## Результаты анализа

### Размер выборок

| Группа | Количество заказов |
|--------|-------------------|
| Control | 280 |
| Test | 320 |

Разница: **40 заказов**

### Стандартное отклонение

| Группа | std |
|--------|-----|
| Control | 11.66 |
| Test | 13.82 |

### Результат t-теста

$$ p\text{-value} = 0.00052 < 0.05 $$

Различия **статистически значимы**

### Изменение среднего времени доставки

$$ \frac{\text{test.mean}}{\text{control.mean}} - 1 \times 100\% = -12.88\% $$

Время доставки в тестовой группе **снизилось на 12.88%**

## Вывод для бизнеса

> Новый алгоритм **статистически значимо** улучшил время доставки (p-value = 0.00052).  
> Среднее время доставки снизилось на **12.88%**.
