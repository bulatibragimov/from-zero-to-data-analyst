# Корреляция и регрессия — анализ связей между переменными

## Теория

### Корреляция

Коэффициент корреляции (Пирсона) измеряет **силу и направление линейной связи** между двумя переменными.

$$ r = \frac{\sum (x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum (x_i - \bar{x})^2 \sum (y_i - \bar{y})^2}} $$

| Значение r | Интерпретация |
|------------|---------------|
| 1.0 | Идеальная положительная связь |
| 0.5 – 0.7 | Сильная положительная |
| 0.3 – 0.5 | Умеренная положительная |
| 0.1 – 0.3 | Слабая положительная |
| 0 | Связи нет |
| -1.0 | Идеальная отрицательная связь |

### Линейная регрессия

Показывает связь между переменными **в изначальных единицах измерения**.

$$ y = \beta_0 + \beta_1 \cdot x + \varepsilon $$

| Компонент | Описание |
|-----------|----------|
| $\beta_0$ | intercept (свободный член) |
| $\beta_1$ | slope (коэффициент наклона) — изменение y при изменении x на 1 |
| $\varepsilon$ | ошибка (то, что не объяснено моделью) |

### Важные предупреждения

| Ловушка | Что это | Как избежать |
|---------|---------|---------------|
| **Корреляция ≠ причинность** | Связь не означает, что A вызывает B | Всегда думать о логике, проверять гипотезы |
| **Гетероскедастичность** | Дисперсия ошибки непостоянна | Визуализировать остатки, использовать robust стандартные ошибки |
| **Нелинейные связи** | Линейная модель не подходит | Рисовать графики, пробовать трансформации |
| **Выбросы** | Искажают корреляцию | Анализировать, удалять или использовать устойчивые методы |

### Классический пример (известная ошибка)

> **Anscombe's quartet** — 4 набора данных с одинаковыми корреляциями и регрессиями, но совершенно разными графиками.

**Главный вывод:** всегда рисуйте свои данные!

## Коэффициенты корреляции

| Метод | Тип данных | Когда использовать |
|-------|------------|---------------------|
| **Пирсона** | Числовые, нормальные | Линейная связь, нет выбросов |
| **Спирмена** | Порядковые / не нормальные | Монотонная связь, устойчив к выбросам |
| **Кендалла** | Порядковые / маленькие выборки | Устойчивый, для малых данных |

```python
# Коэффициенты корреляции в Python
import scipy.stats as sts

# Пирсона
sts.pearsonr(x, y)

# Спирмена
sts.spearmanr(x, y)

# Кендалла
sts.kendalltau(x, y)
```

---

## Линейная регрессия в Python

```python
from scipy.stats import linregress

# Простая линейная регрессия
result = linregress(x, y)

# Доступные атрибуты
result.slope       # коэффициент наклона (β₁)
result.intercept   # свободный член (β₀)
result.rvalue      # коэффициент корреляции
result.pvalue      # p-value для slope
result.stderr      # стандартная ошибка
```

## Диагностика регрессии

### Что проверять

| Диагностика | Что смотреть | Как улучшить |
|-------------|--------------|---------------|
| **Линейность** | scatter plot (y ~ x) | Трансформация переменных |
| **Нормальность остатков** | QQ-plot, гистограмма | Больше данных, bootstrap |
| **Гомоскедастичность** | residuals vs fitted | Robust стандартные ошибки |
| **Отсутствие выбросов** | Cook's distance, leverage | Удалить/изолировать |

### Код для остатков

```python
import matplotlib.pyplot as plt

# Предсказанные значения и остатки
predicted = result.intercept + result.slope * x
residuals = y - predicted

# График остатков
plt.scatter(predicted, residuals)
plt.axhline(y=0, color='r', linestyle='--')
plt.xlabel('Предсказанные значения')
plt.ylabel('Остатки')
plt.show()
```

## Практические задачи (решены)

### Задача 1: Связь между ростом и весом

- **Метод:** корреляция Пирсона
- **Результат:** сильная положительная связь (r ≈ 0.85)

### Задача 2: Влияние рекламного бюджета на продажи

- **Метод:** линейная регрессия
- **Результат:** увеличение бюджета на 1 тыс. руб. → рост продаж на 2.3 тыс. руб.

### Задача 3: Анализ зависимости цены от возраста авто

- **Метод:** корреляция Спирмена (данные не нормальные)
- **Результат:** сильная отрицательная связь (чем старше авто, тем дешевле)

### Задача 4: Проверка гетероскедастичности

- **Проблема:** дисперсия остатков растёт с предсказанными значениями
- **Решение:** использовать логарифмирование зависимой переменной

### Задача 5: Обнаружение нелинейной связи

- **Проблема:** корреляция Пирсона ≈ 0, но связь есть (квадратичная)
- **Урок:** всегда рисовать графики! (пример Anscombe)

## Визуализация связей

| График | Что показывает |
|--------|----------------|
| **Scatter plot** | Исходные точки и форму связи |
| **regplot** (seaborn) | Scatter + линия регрессии + доверительный интервал |
| **Joint plot** | Scatter + гистограммы + корреляция |
| **Heatmap корреляций** | Матрица корреляций между всеми переменными |

```python
import seaborn as sns

# Scatter с линией регрессии
sns.regplot(x=x, y=y)

# Матрица корреляций
sns.heatmap(df.corr(), annot=True, cmap='coolwarm')
```
