# Какие навыки приобретем?
- быстро и эффективно работать с таблицами 
- писать sql-запросы с фильтрацией и группировкой
- объединять таблицы с помощью join
- составлять сложные запросы с подзапросами и оконными функциями
- расчитывать продуктовые метрики и решать реальные бизнес задачи 

## С какими данными работает аналитик?
- ЧТО: *Данные* - это зафиксированные факты о событиях, действиях, объектах.
- ГДЕ: бизнес, наука, медицина, обраховани и тд
- ЗАЧЕМ: 1) понять поведение на текущий момент 2) улучшать процессы 3) принимать решения на основе фактов

## С какими интсрументами работает аналитик?
1. Электронные таблицы
>Небольшие объемы данных
2. SQL (Structured Query Language )
>Язык запросов к базам данных. 
>Позволяет выбирать, фильтровать, объединять и агрегировать данные.
>Основа работы аналитика с хранилищами данных

3. Python
> Универсальный язык для анализа и автоматизации.
Позволяет работать с большими данными, строить модели, визуализировать.
Используется для продвинутой аналитики и машинного обучения
4. ВI-системы (Power BI, Tableau, Looker и др.) 
> Инструменты для создания интерактивных дашбордов и визуализаций.
Помогают бизнесу отслеживать ключевые метрики в реальном времени.
Упрощают принятие решений на основе данных
## Зачем аналитику SQL?
- Доступ к "сырым" данным без искажений и агрегаций;
- Гибкость при анализе: можно строить любые выборки и срезы;
- Возможность обрабатывать  большие объёмы данных;
- Основа для продвинутой аналитики – моделей, дашбордов, А/В-тестов.
