# SQL Analytics Portfolio

## Overview

Этот раздел репозитория содержит практические задачи и проекты по SQL-аналитике данных.

Материалы основаны на учебном курсе **Karpov.Courses SQL** и демонстрируют ключевые навыки, необходимые для работы аналитиком данных.

Основная цель этого раздела — показать умение:

* работать с реляционными базами данных
* анализировать бизнес-данные с помощью SQL
* писать читаемые и оптимизированные запросы
* решать реальные аналитические задачи

---

# Topics Covered

В этом разделе представлены следующие темы:

* SQL Basics
* Data Filtering
* Aggregation and Grouping
* Table Joins
* Subqueries
* Window Functions
* Query Optimization

# Skills Demonstrated

В этом разделе демонстрируются следующие навыки SQL-аналитики:

* фильтрация данных (`WHERE`)
* агрегации (`COUNT`, `SUM`, `AVG`)
* группировка (`GROUP BY`, `HAVING`)
* объединение таблиц (`JOIN`)
* подзапросы (`Subqueries`)
* оконные функции (`ROW_NUMBER`, `RANK`, `LAG`, `LEAD`)
* оптимизация SQL-запросов
* анализ пользовательского поведения
